# Project Setup

- Install the stable version of [NodeJS](https://nodejs.org/en/) and npm.
  
 
 
# Running the app 
   
   To locally run the application,
   
   - Clone this repository.
   - To install depedencies in package.json, npm install. 
   - Run react project using `npm start`.
